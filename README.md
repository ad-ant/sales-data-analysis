# Sales Data Analysis

This project presents an exploratory analysis of sales data using Python, Pandas, and SQL, with a focus on deriving business insights.

## Scope

* Loading and exploring structured sales data
* Performing time-based aggregations and trend analysis
* Analyzing sales performance across products, categories, and countries
* Using SQL (SQLite) for data querying and aggregation
* Interpreting results to extract actionable business insights

## Technologies

* Python
* Pandas
* SQLite
* Jupyter Notebook

## Key Insights

* Sales vary across months, indicating seasonal patterns
* Electronics generate the highest overall revenue
* Poland and Germany are the strongest markets in terms of sales

## Notes

The project demonstrates a complete data analysis workflow, from data exploration and transformation to insight generation, with a focus on practical business understanding.
